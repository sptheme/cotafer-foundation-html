Manage layout

You can manage layout by adding class into body tag by use one of these classes:

  • (default: right-sidebar)
  • left-sidebar: Float sidebar to left of primary content
  • full-screen: Allow content's page display fullscreen base browser's size.
  • full-width: Allow content's page display full-width base on .container's width
